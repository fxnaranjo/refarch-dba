## Digital business automation architecture

The current digigtal business automation architecture is documented in the [IBM Cloud architectures](https://www.ibm.com/cloud/architecture/architectures/dba-architecture). 

* [Workflow management architecture](https://www.ibm.com/cloud/architecture/architectures/workflowDomain) includes runtime and business process authoring components in a single view. 
* [Content management architecture](https://www.ibm.com/cloud/architecture/architectures/contentManagementdomain) defines the capabilities and tools to secure, govern, understand, and automate access to content.
* [Business decision automation architecture](https://www.ibm.com/cloud/architecture/architectures/businessDecisionAutomationDomain)
* [Robotic process automation architecture](https://www.ibm.com/cloud/architecture/architectures/businessDecisionAutomationDomain)
* [Document data capture architecture](https://www.ibm.com/cloud/architecture/architectures/captureDomain)
* [Operational intelligence architecture](https://www.ibm.com/cloud/architecture/architectures/operationalIntelligenceArchitecture)
