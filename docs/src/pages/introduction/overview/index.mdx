## Introduction 

This repository includes working content on best practices and reusable content for deploying business application using IBM Cloud Pak for Automation products.

In this repository we are developing new content for integrating cloud native application, DevOps practices and integrating new DBA components. 

## Architecture center

The reference architecture is described in the [IBM reference architecture center](https://www.ibm.com/cloud/architecture/architectures/dba-architecture) and the content reflects standard capabilities to consider for deploying solution.
